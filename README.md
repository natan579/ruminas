# Ruminas - Multi-Page Web Application

A modern, responsive multi-page web application built with HTML, CSS, and JavaScript.

## 📁 Project Structure

```
ruminas/
├── index.html           # Home page
├── pages/
│   ├── about.html       # About page
│   ├── services.html    # Services page
│   └── contact.html     # Contact page
├── styles/
│   └── style.css        # Main stylesheet
├── js/
│   └── script.js        # JavaScript functionality
└── README.md            # Project documentation
```

## 🚀 Features

- **Responsive Design**: Mobile-first approach with breakpoints at 768px and 480px
- **Navigation**: Sticky navigation bar with mobile hamburger menu
- **Multi-Page**: Four main pages (Home, About, Services, Contact)
- **Contact Form**: Functional contact form with validation
- **Modern Styling**: Gradient backgrounds, smooth animations, and hover effects
- **Interactive Elements**: Smooth scrolling, form handling, and menu toggle

## 📄 Pages

### Home (index.html)
- Hero section with call-to-action button
- Features showcase with 4 feature cards
- Professional footer with social links

### About (pages/about.html)
- Company story and mission
- Core values section
- Team information

### Services (pages/services.html)
- 6 service offerings with icons
- Service descriptions
- Grid layout for easy browsing

### Contact (pages/contact.html)
- Contact information (address, phone, email, hours)
- Contact form with validation
- Form submission feedback

## 🎨 Design Features

- **Color Scheme**: Purple gradients (#667eea to #764ba2) with gold accents (#ffd700)
- **Typography**: Segoe UI font family for modern appearance
- **Animations**: Smooth transitions and slide-in animations
- **Responsive Grid**: CSS Grid and Flexbox for flexible layouts

## 💻 JavaScript Functionality

- Mobile menu toggle
- Navigation link highlighting
- Form validation and submission
- Email format validation
- Success/error message display
- Smooth scroll behavior

## 📱 Responsive Breakpoints

- **Desktop**: Full layout (1200px+)
- **Tablet**: Adjusted layout (768px)
- **Mobile**: Hamburger menu and stacked layout (480px)

## 🔧 How to Use

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Navigate through the pages using the navigation menu
4. Test the contact form functionality
5. Customize content and styling as needed

## 📝 Customization

To customize the project:

1. **Update Content**: Edit text in HTML files
2. **Change Colors**: Modify gradient colors in `style.css`
3. **Add Pages**: Create new HTML files in the `pages/` directory
4. **Modify Navigation**: Update nav-links in the navbar
5. **Form Handling**: Integrate with backend service in `script.js`

## 🌐 Contact Form Integration

The contact form currently simulates submission. To integrate with a real backend:

1. Update the `handleFormSubmit()` function in `script.js`
2. Send form data to your backend API
3. Handle server responses appropriately
